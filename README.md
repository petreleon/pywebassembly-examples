# pywebassembly-examples

Acest repository conține exemple și exerciții pentru proiectul `pywebassembly`.

## Structura Proiectului

### `01_exercitii_elementare`
Acest director conține o serie de exerciții de programare elementare, definite în format JSON. Fiecare fișier JSON conține descrierea problemei, datele de intrare/ieșire pentru teste și un șablon de cod Python.

Lista exercițiilor incluse:
1.  `01_pozitiv_negativ_zero.json` - Verificarea unui număr (pozitiv, negativ sau zero).
2.  `02_par_sau_impar.json` - Verificarea parității unui număr.
3.  `03_cine_este_mai_mare.json` - Compararea a două numere.
4.  `04_calificative_scolare.json` - Transformarea punctajelor în calificative.
5.  `05_cel_mai_mare_dintre_trei.json` - Determinarea maximului dintre trei numere.
6.  `06_eligibil_angajare.json` - Verificarea eligibilității pentru angajare.
7.  `07_anul_bisect.json` - Verificarea anilor bisecți.
8.  `08_tipul_de_triunghi.json` - Determinarea tipului de triunghi pe baza laturilor.
9.  `09_ecuatia_de_gradul_2.json` - Rezolvarea ecuației de gradul 2.
10. `10_sistem_complex_reduceri.json` - Calculul prețului final cu reduceri complexe.
